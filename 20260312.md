>
> You are given an integer sequence A=(A1,A2,…,AN) of length N.
>
> Find the maximum length of a subsequence B=(B1,B2,…,B∣B∣) of integer sequence A that satisfies the following condition.
>
> - Bi+1=Bi+1 for all integers i satisfying 1≤i≤∣B∣−1.
>

​	或者可以这样：我们可以在进行每一步输入的时候都进行计算，创建一个vector<int>来存放已经统计了的，如果有重复的，即取最大值。实现起来不困难，代码如下：

```c++

#include <vector>
using namespace std;

    cin.tie(nullptr);
    cout.tie(nullptr);

    int n, top = 0;
    cin >> n;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;

        top = max(top, v[x]);
    }
    cout << top << '\n';
    return 0;

```

​	不对，怎么Re了...观察到其实是An < 1e9, 而vector没这么大。此时有一位强者告诉我可以用unordered_map来实现。

​	unordered_map，C++中stl里的一个无序容器，使用时与数组差异不大，是存储``键值对``的无序容器，核心优势是**O (1) 级别的快速查找 / 插入**（基于哈希表（Hash Table）），并且可以实现动态分配，下标任意可哈希类型，但代价是内存开销大。

代码：

```c++
#include <iostream>
#include <unordered_map>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);
    
    unordered_map<int,int> v;
    int n, top = 0;
    cin >> n;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        v[x] = max(v[x], v[x - 1] + 1);
        top = max(top, v[x]);
    }
    cout << top << '\n';
    return 0;
}
```

### [F - Center Alignment](https://atcoder.jp/contests/abc445/tasks/abc445_b)

> ### Problem Statement
>
> You are given N strings S1,S2,…,SN of odd lengths consisting of lowercase English letters.
>
> Let m be the length of the longest string among S1,S2,…,SN. Find strings T1,T2,…,TN satisfying the following condition.
>
> - Condition: Ti is a string of length m formed by concatenating k copies of `.`, Si, and k copies of `.` in this order, for some non-negative integer k.
>
> ### Constraints
>
> - N is an integer between 1 and 100, inclusive.
> - Si is a string of odd length between 1 and 99, inclusive, consisting of lowercase English letters.
>
> ------
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N
> S1
> S2
> ⋮
> SN
> ```
>
> ### Output
>
> Output N lines. The i-th line (1≤i≤N) should contain Ti.

#### 分析：

​	我们可以先输入所有字符串，在输入的过程中用String.length()来获取长度，自动得出每个的长度和最大长度，并由此来计算出所需填补的`.`的个数。

#### 代码：

```c++
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int n;
    cin >> n;

    vector<string> v(100);
    vector<int> v1(100);
    int max = 0;
    for (int i = 0; i < n; i++) {
        cin >> v[i];
        v1[i] = v[i].length();
        if (v1[i] > max) max = v1[i];
    }

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < (max - v1[i]) / 2; j++) {
            cout << ".";
        }
        cout << v[i];
        for (int j = 0; j < (max - v1[i]) / 2; j++) {
            cout << ".";
        }
        cout << '\n';
    }
    return 0;
}
```

### [G - Sugoroku Destination](https://atcoder.jp/contests/abc445/tasks/abc445_c)

> ### Problem Statement
>
> There are N cells, cell 1, cell 2,…, cell N, arranged in a line. Cell i has an integer Ai (i≤Ai≤N) written on it.
>
> For each of s=1,2,…,N, solve the following problem.
>
> - Initially, place a piece on cell s. After performing the operation "let x be the integer written on the cell where the piece is placed, and then move the piece to cell x" 10100 times, output the number of the cell where the piece is placed.
>
> ### Constraints
>
> - 1≤N≤5e5
> - i≤Ai≤N (1≤i≤N)
> - All input values are integers.
>
> ------
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N
> A1 A2 … AN
> ```
>
> ### Output
>
> Output the answers for s=1,2,…,N in this order on a single line, separated by spaces.

分析：

​	题意为以s点开始，移动到当前位置所表示的数的格，循环这个步骤1e100次。走 1e100 步等价于走到不再动为止。

​	我们可以在输入Ai时预处理其将要前往的格子，以vector~~(已经快变成vector的形状了)~~来表示。

​	此时代码如下：

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int n;
    cin >> n;
    vector<int> v(n);
    int temp;
    for (int k = 0; k < n; k++) {
        cin >> temp;
        v[k] = temp;
    }
    for (int i = 0; i < n; i++) {
        int flag = 0, location = i + 1, last = 0;
        for (int j = 0; flag != 2; j++) {
            location = v[location];
            if (location == last) {
                flag ++;
            }
            else {
                flag = 0;
            }
        }
        cout << location;
        if (i < n - 1) {
            cout << " ";
        }
    }
    return 0;
}
```

​	~~不是今天怎么老超时~~ 又一次tle了，说明还可以优化预处理。我们不妨进行如下预处理：

​	1)查找是否有输入的Ai所指代的位置。如果有，则直接最后的值。

​	2)检查是否An与n本身循环相等，如果相等则说明已经走完，记录n为终点值。

​	3)递归继续计算终点值。

```c++
vector<int> end(500005, 0);
vector<int> a(500005, 0);
    
int finding(int n){
    if (end[n] != 0){
        return end[n];// 之前已经计算过，直接返回
    }
    if (a[n] == n){
        end[n] = n;
        return n;// 循环了，得出最后值
    }
    return end[n] = finding(a[n]);
}
```

代码：

```c++
#include <iostream>
#include <vector>
using namespace std;

vector<int> ending(500005, 0);
vector<int> a(500005, 0);

int finding(int n){
    if (ending[n] != 0){
        return ending[n];// 之前已经计算过，直接返回
    }
    if (a[n] == n){
        ending[n] = n;
        return n;// 循环了，得出最后值
    }
    return ending[n] = finding(a[n]);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int n;
    cin >> n;

    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }

    for (int k = 1; k <= n; k++) {
        cout << finding(k);
        if (k < n) {
            cout << " ";
        }
    }
    return 0;
}
```
