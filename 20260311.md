### [A - Handmaid](https://atcoder.jp/contests/abc446/tasks/abc446_a)

> ### Problem Statement
>
> You are given the name SS of a certain person. The first character of SS is an uppercase English letter, and the other characters are lowercase English letters.
>
> The name of this person's handmaid is the string obtained by converting the first letter of SS to lowercase and adding `Of` to the beginning. Find the name of this handmaid.
>
> ### Constraints
>
> - S is a string of length between 1 and 10, inclusive.
> - The first character of S is an uppercase English letter.
> - The characters of S other than the first are lowercase English letters.
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> S
> ```
>
> ### Output
>
> Output the answer on one line.

#### 分析：

​	本题输入一个字符串，在前面加上`Of`，并把首字母变成小写再输出即可。

#### 代码：

```c++
#include <iostream>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    cout << "Of";

    string S;
    cin >> S;

    bool flag = false;

    for (char a : S) {
        if (!flag) {
            cout << (char)(a + 32);
            flag = true;
            continue;
        }
        cout << a;
    }

    return 0;
}
```

### [B - Greedy Draft](https://atcoder.jp/contests/abc446/tasks/abc446_b)

> ### Problem Statement
>
> There are N customers numbered 1 to N, and M canned juices numbered 1 to M.
>
> Customer i (1≤i≤N) has a wish list of length Li. The j-th item (1≤j≤Li) from the top of customer i's wish list is canned juice Xi,j. For any customer i, the numbers Xi,1,…,Xi,Li on customer i's wish list are distinct.
>
> Customers 1,…,N, in this order, will now choose their beverages, following the procedure below.
>
> - If the customer's wish list contains a canned juice that has not yet been chosen by anyone at that point, they choose the canned juice whose number appears earliest in their wish list. Otherwise, they choose  water.
>
> Determine which beverage each customer gets.
>
> ### Constraints
>
> - 1≤N≤100
> - 1≤M≤100
> - 1≤Li≤M (1≤i≤N)
> - 1≤Xi,j≤M (1≤i≤N, 1≤j≤Li)
> - Xi,1,…,Xi,Li are distinct. (1≤i≤N)
> - All input values are integers.
>
> ------
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N M
> L1
> X1,1 X1,2 ⋯ X1,L1
> L2
> X2,1 X2,2 ⋯ X2,L2
> ⋮
> LN
> XN,1 XN,2 ⋯ XN,LN
> ```
>
> ### Output
>
> Output N lines. The i-th line (1≤i≤N) should contain the number of the canned juice customer i gets if they get one, or `0` if customer i gets water.

#### 分析：

​	一共有N个人，需要饮料，根据所剩余的愿望单中最高的项目拿取饮料。可以先定义一个长数列（因为m不超过100），通过从其中按照顺序检索其愿望单来实现。

#### 代码：

```c++
#include <iostream>
using namespace std;
const int MAXNUM = 105;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int N, M;
    cin >> N >> M;

    int drinks[MAXNUM] = {0};
    for (int i = 1; i <= M; i++) {
        drinks[i] = 1;
    }
    int L, select, ans;
    bool flag;
    for (int n = 0; n < N; n++) {
        cin >> L;
        flag = false;
        for (int m = 0; m < L; m++) {
            cin >> select;
            if (drinks[select] == 1 && flag == false) {
                drinks[select] --;
                ans = select;
                flag = true;
            }
        }
        if (!flag) {
            cout << 0 << '\n';
            continue;
        }
        cout << ans << '\n';
    }
    return 0;
}
```

> - In the evening of day i, all eggs that have been stocked for D or more days are discarded.
>
> There were no eggs in stock before the morning of day 1, and eggs never ran out at noon on any day.
>
> Find how many eggs remain in the restaurant after the evening action on day N.
>
> T test cases are given; solve each.
>
> ### Constraints

> - 1≤T≤2×105
> - 1≤D≤N≤2×105
> - 1≤Ai,Bi≤10

#### 代码：

```c++
#include <iostream>
#include <queue>
using namespace std;

void function();

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int N;
    cin >> N;
    for (int i = 0; i < N; i++) {
        function();
    }

    return 0;
}

void function() {
    queue <pair<int, int>> q;
    int n, d;
    cin >> n >> d;
    for (int i = 1; i <= n; i++) {
        int a;
        cin >> a;
        q.emplace(a, i);
    }
    for (int i = 1; i <= n; i++) {
        int temp;
        cin >> temp;// 今日使用

        while (temp != 0) {
            if (q.front().first <= temp) {
                temp -= q.front().first;
                q.pop();
            }
            else {
                q.front().first -= temp;
                temp = 0;
            }// 此处由于规定了所用鸡蛋数不会超过当前的鸡蛋数，所以不用考虑超过的情况
        }

        while (!q.empty() && q.front().second <= i - d) {
            q.pop();
        }
    }
    int sum = 0;
    while (!q.empty()) {
        sum += q.front().first;
        q.pop();
    }
    cout << sum << '\n';
}
```

### [E - Strong Word](https://atcoder.jp/contests/abc445/tasks/abc445_a)

> ### Problem Statement
>
> You are given a string S of length at least 2 consisting of lowercase English letters.
>
> Determine whether the first character and the last character of S are the same.
>
> ### Constraints
>
> - S is a string of length between 2 and 10, inclusive, consisting of lowercase English letters.
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> S
> ```
>
> ### Output
>
> If the first character and the last character of S are the same, output `Yes`; otherwise, output `No`.

#### 分析：

​	本题输入一个字符串，判断第一个字符和最后一个字符是否相同。

#### 代码：

```c++
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    char first, end, gain;
    first = getchar();
    while (true) {
        gain = getchar();
        if (gain == '\n') {
            break;
        }
        end = gain;
    }
    if (first == end) {
        cout << "Yes";
        return 0;
    }
    cout << "No";
    return 0;
}
```
