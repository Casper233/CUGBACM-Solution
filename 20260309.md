### [A - chmin](https://atcoder.jp/contests/abc448/tasks/abc448_a)

> ### Problem Statement
>
> You are given a length-NN integer sequence A=(A1,A2,…,AN)A=(A1,A2,…,AN) and an integer XX.
> For i=1,2,…,Ni=1,2,…,N in this order, do the following.
>
> - If Ai<XAi<X, update X=AiX=Ai and output `1`.
> - Otherwise, output `0`.
>
> ### Constraints
>
> - All input values are integers.
> - 1≤N,X,Ai≤1001≤N,X,Ai≤100
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N X
> A1 A2 …… An
> ```
>
> ### Output
>
> Output N lines.
> The k-th line should contain the output for i=k.

#### 分析：

​	题目输入N, X。我们需要比较输入的数An和X：如果比X小，则输出1并将X改为An；反之则只输出0。

​	每行输出一个结果。

​	可以输入N和X之后执行for循环，在每次循环中比较An与X。

#### 代码：

```c++
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int n, x, temp = 0;
    cin >> n >> x;
    for (int i = 0; i < n; i++) {
        cin >> temp;
        if (temp < x) { //当An比X小时
            x = temp;
            cout << 1 << "\n";
        }
        else { //当An比X大时
            cout << 0 << "\n";
        }
    }
    return 0;
}
```

### [B - Pepper Addiction](https://atcoder.jp/contests/abc448/tasks/abc448_b)

> ### Problem Statement
>
> Takahashi loves pepper.
>
> A restaurant has M types of pepper, called type 1,2,…,M. There are Cj grams of type j (1≤j≤M) pepper at this restaurant.  
>
> He ordered N dishes at this restaurant.
> Due to compatibility, only type Ai​ pepper can be sprinkled on dish i (1≤i≤N), and the upper limit on the amount of pepper that can be sprinkled on dish i is Bi​ grams.
> Also, he can only use the pepper available at the restaurant. That is, the total amount of type j pepper sprinkled cannot exceed Cj​ grams.
>
> He decides how much pepper to sprinkle on each dish to maximize the total amount of pepper sprinkled on the dishes.
> How many grams of pepper in total can he sprinkle on the dishes?
>
> ### Constraints
>
> - All input values are integers.
> - 1≤N,M≤1000
> - 1≤Ai≤M
> - 1≤Bi,Ci≤106
>
> ------
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N M
> C1 C2 … Cm
> A1 B1
> A2 B2
> ⋮
> An Bn
> ```
>
> ### Output
>
> Output the answer as an integer.

#### 分析：

​	首先输入N餐品数和M辣椒品种数，并在下一行输入每个品种的克数。接下来n行分别输入所需品种和最大克数。最后输出总使用的克数。

​	我们可以先新建一个数列用来计算当前各品种辣椒的剩余克数，并在每次输入后计算剩余量，并求和使用量。

#### 代码：

```c++
#include <iostream>
using namespace std;

const int MAXNUM = 1005;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);
    
    int N, M, Pepper[MAXNUM] = {0};
    cin >> N >> M;
    for (int i = 0; i < M; i++) {
        cin >> Pepper[i];
    }
    int a, b, total = 0;
    for (int i = 0; i < N; i++) {
        cin >> a >> b;
        if (Pepper[a - 1] == 0) { //处理无当前辣椒的情况
            continue;
        }
        if (Pepper[a - 1] >= b) { //处理当前剩余相等或超过最大量
            total += b;
            Pepper[a - 1] -= b;
            continue;
        }
        //处理小于
        total += Pepper[a - 1];
        Pepper[a - 1] = 0;
    }
    cout << total;
    return 0;
}
```

### [C - Except and Min](https://atcoder.jp/contests/abc448/tasks/abc448_c)

> ### Problem Statement
>
> A bag contains N balls numbered 1 to N. Ball i has the integer Ai written on it.
>  Process Q queries.
>  For each query, a sequence B1​,B2​,…,BK​ of length K is given, and you should perform the following sequence of operations. Here, all Bi​ are between 1 and N inclusive and are distinct.
>
> - First, remove balls B1, B2, …, BK from the bag.
> - Then, output the minimum value among the integers written on the  balls currently in the bag. (It is guaranteed by the constraints that  the bag is not empty at this point.)
> - Afterwards, return all K removed balls to the bag.
>
> ### Constraints
>
> - 6≤N≤3×105
> - 1≤Q≤2×105
> - 1≤Ai≤109
> - 1≤K≤5
> - 1≤B1<B2<⋯<Bk≤N
> - The sum of K over all queries is at most 4×105.
> - All input values are integers.
>
> ------
>
> ### Input
>
> The input is given from Standard Input in the following format, where queryi denotes the i-th query.
>
> ```
> N Q
> A1 A2 … AN
> query1
> query2
> ⋮
> queryQ
> ```
>
> Each query is given in the following format.
>
> ```
> K
> B1 B2 … BK
> ```
>
> ### Output
>
> Output Q lines. The i-th line should contain the answer to the i-th query.

#### 分析：

​	有一个袋子，里面装着编号从 1 到 N 个球。现在要处理 Q 个查询，且遵循以下过程：

​		1.先从袋子里取出编号为 B1、B2、……、Bk 的球。

​		2.接着，输出袋子里剩下球上整数的最小值。

​		.最后，把刚才取出的所有 KK 个球放回袋子。

​	所以我们可以先创建一个数列，然后取出所有被拿出的球，再遍历。每次循环可以用以下方式完成：

```c++
cin >> K;
int Pull[K];// 取出的K个球
for (int j = 0; j < K; j++) {
   cin >> Pull[j];// 按顺序所以无需排序
}
min = 0, count = 0;
for (int j = 0; j < N; j++) {
    if (j == Pull[count]) {// 取出来的球，跳过此次for循环
        count += 1;
        continue;
    }
	if (Balls[j] < min) {// 非取出来的小于当前min的球
		min = Balls[j];
	}
}
cout << min << '\n';
```

#### 代码：

```C++
#include <algorithm>
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q;
    cin >> N >> Q;
    vector<pair<int, int>> A(N);// 建立二维vector
    for (int i = 0; i < N; i++) {
        cin >> A[i].first;
        A[i].second = i;
    }// 将其大小计入第一个元素，顺序计入第二个元素
    sort(A.begin(), A.end());// 按照大小排序
    for (int i = 0; i < Q; i++) {
        int K;
        cin >> K;
        vector<int> B(K);// 定义第二个只含5个元素的vector
        for (int j = 0; j < K; j++) {
            cin >> B[j];
        }
        for (int j = 0; j < N; j++) {
            if (find(B.begin(), B.end(), A[j].second + 1) != B.end()) { // 当二者元素相同时，继续for循环
                continue;
            }
            cout << A[j].first << '\n';// 元素不同时 得出最小的An
            break;
        }
    }
    return 0;
}
```

### [E - Seats 2](https://atcoder.jp/contests/abc447/tasks/abc447_a)

> ### Problem Statement
>
> There are NN seats arranged in a row. Each seat can accommodate at most one person. Determine whether it is possible to seat MM people in the seats satisfying the following condition.
>
> > No two adjacent seats may both be occupied.
>
> ### Constraints
>
> - 1≤N,M≤1001≤N,M≤100
> - All input values are integers.
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> NN MM
> ```
>
> ### Output
>
> Output `Yes` if it is possible to seat the people satisfying the condition, and `No` otherwise.

#### 分析：

​	相邻的两把椅子不能都被占用。即3把椅子可坐最多2人，4把椅子也是最多两人。

​	即椅子数被2整除，如果有余数则可以坐/2+1人，没有则只能做/2人。

#### 代码：

```c++
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int n, m;
    cin >> n >> m;
    if (n % 2 != 0) {
        n ++;
    }
    if (n / 2 < m) {
        cout << "No";
        return 0;
    }
    cout << "Yes";
    return 0;
}
```
