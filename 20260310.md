### [D - Integer-duplicated Path](https://atcoder.jp/contests/abc448/tasks/abc448_d)

> ### Problem Statement
>
> You are given a tree with NN vertices numbered 1,2,…,N1,2,…,N. The i-th edge connects vertices Ui and Vi.
> Each vertex ii has an integer Ai​ written on it.
>
> Answer the following problem for all k=1,2,…,N.
>
> - Problem: Among the vertices on the simple path (a path that does not visit the same vertex more than once) from vertex 11 to vertex kk, if there exist two distinct vertices with the same integer written on them, output `Yes`; otherwise, output `No`.
>   - It can be proved that the simple path connecting two vertices on a tree is unique.
>
> ### Constraints
>
> - All input values are integers.
> - 2≤N≤2e5
> - 1≤Ai≤1e9
> - 1≤Ui,Vi≤N
> - The given graph is a tree.
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> N
> A1 A2 …… AN
> U1 V1
> U2 V2
> ⋮   ⋮
> Un−1 Vn−1
> ```
>
> ### Output
>
> Output N lines.
>  The i-th line should contain the answer to the problem for k=i.

#### 分析：

​	通过画图不难看出，其为线段树问题。通过对其父节点和本身的值进行判断，只要有任何值有重复即可输出Yes，否则输出No。可以使用dfs来解决。

​	首先我们可以创建一个邻接表（矩阵），来记录整个树。

```c++
vector<int> A(N + 1);
for (int i = 0; i < N; i++) {
    cin >> A[i];
}// 记录An

vector<vector<int>> Connect(N + 1);// 建立邻接表
for (int i = 0; i < N; i++) {
    int u, v;
    cin >> u >> v;
    Connect[u].push_back(v);
    Connect[v].push_back(u);// 由于是无方向树，所以还要加上其父节点
}
```

​	之后从1出发，遍历整个树，在遍历的过程中比较此时的An是否已经出现过。由于Ai较大，使用数组内存开销较大，可以使用stl的map（映射）。

​	在一次遍历中，我们便可以得出所有答案（每次跳进一层或跳回最近分支都可看做k+1）。

​	每次的过程如下：

​	1)进入节点，判断此时所代表的数是否曾经出现过

​	2)将1)答案赋值给输出vector

​	3)记录此节点的An

​	4)找下一个相连的非父节点

​	5)回撤，删除此节点记录的An

```c++
void dfs(int u, int father, int dup_status) {
    if (M[A[u]] != 0) {
        dup_status = 1;// 进入节点，判断此时所代表的数是否曾经出现过
    }
    Output[u] = dup_status; //将答案赋值给输出vector

    M[A[u]] ++;// 记录此节点的An
    for (int v : Connect[u]) {
        if (v == father) {
            continue;
        }
        dfs(v, u, dup_status);// 找下一个相连的非父节点
    }
    M[A[u]] --;// 回撤，删除此节点记录的An
}
```

​	组合拼凑一下，再注意相关for循环的起始位置和条件，就可以拼出这道题的代码。

#### 代码：

```c++
#include <iostream>
#include <vector>
#include <map>
using namespace std;

vector<int> A;
vector<vector<int>> Connect;
vector<int> Output;
map<int, int> M;

void dfs(int u, int father, int dup_status);

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int N;
    cin >> N;
    A.resize(N + 1);
    for (int i = 1; i <= N; i++) {
        cin >> A[i];
    }
    Connect.resize(N + 1);
    for (int i = 0; i < N - 1; i++) {
        int u, v;
        cin >> u >> v;
        Connect[u].push_back(v);
        Connect[v].push_back(u);
    }
    Output.resize(N + 1, 0);

    dfs(1, 0, 0);

    for (int i = 1; i <= N; i++) {
        if (Output[i]) {
            cout << "Yes" << "\n";
        }
        else {
            cout << "No" << "\n";
        }
    }
    return 0;
}

void dfs(int u, int father, int dup_status) {
    if (M[A[u]] != 0) {
        dup_status = 1;
    }
    Output[u] = dup_status;

    M[A[u]] ++;
    for (int v : Connect[u]) {
        if (v == father) {
            continue;
        }
        dfs(v, u, dup_status);
    }
    M[A[u]] --;
}
```

### [F - mpp](https://atcoder.jp/contests/abc447/tasks/abc447_b)

> ### Problem Statement
>
> You are given a string S consisting of lowercase English letters. Remove all occurrences of the most frequent character(s) in S, then output the remaining characters concatenated in their original order.
>
> If there are multiple characters with the maximum frequency, remove all of them.
>
> ### Constraints
>
> - 1≤∣S∣≤100
> - S is a string consisting of lowercase English letters.
>
> ### Input
>
> The input is given from Standard Input in the following format:
>
> ```
> S
> ```
>
> ### Output
>
> Output the answer.

#### 分析：

​	我们需要统计原来的字符串S的各个字符的顺序，并统计数量，删去重复数目最高的。

​	我们可以使用一个字符串来记录S，遍历一次来得出最多的字符（可以用vector也可以用int a[]），再for循环进行输出，同时不处理数量最多的字符即可。

#### 代码：

```c++
#include <string>
#include <vector>
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    vector<int> count(26, 0);// vector需要设置大小，否则下面的++会爆炸的（
    for (char a : S) {
        count[a - 'a'] ++;// 输入字符串
    }
    int maxCount = 0;
    for(int i : count) {
        if (i > maxCount) {// 遍历其中数目最多的
            maxCount = i;
        }
    }
    for (char i : S) {
        if (count[i - 'a'] != maxCount) {// 排除数目最多的
            cout << i;
        }
    }
    return 0;
}
```
